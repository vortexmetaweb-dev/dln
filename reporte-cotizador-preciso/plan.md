## Key Arguments / Thesis
- El archivo actual es un catálogo fiscal-operativo, no una matriz tarifaria.
- El vacío de moneda y precio impide una cotización precisa.
- El cotizador debe separar catálogo de conceptos, matriz tarifaria y parámetros de operación.
